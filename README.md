# PowerBI_dashbord_Social_Media_Addiction
This project presents an interactive Power BI dashboard that analyzes patterns of social media usage and its impact on addiction levels and mental health among students.  The dashboard provides a comprehensive view of how different factors such as country, gender, academic level, and most-used platforms influence social media addiction.

Data Source
📁 Dataset: Social Media Addiction Dataset
🌐 Source: Kaggle

📊 Data includes:
Student ID
Age
Gender
Country
Academic Level
Most Used Platform
Average Daily Usage Hours
Addiction Score
Mental Health Score

Key Features of Dashboard

📌 KPI Cards
Average Daily Usage Hours
Average Addiction Score
Average Mental Health Score

🌍 Country-wise Analysis
Identifies countries with the highest addiction scores

📱 Platform Usage Analysis
Distribution of users across platforms like Instagram, TikTok, Facebook, etc.

👥 Demographic Insights
Comparison based on Gender
Analysis based on Academic Level

🎛️ Interactive Filters (Slicers)
Country
Platform
Gender
Age

💡 Key Insights

📈 High Usage = High Addiction
Users with higher daily usage hours tend to have significantly higher addiction scores.

📱 Platform Dominance
Instagram emerges as the most widely used platform among students, followed by TikTok.

🌍 Country Variation
Certain countries show notably higher addiction levels, indicating regional behavioral patterns.

👩‍🎓 Gender Impact
Female users show a relatively higher mental health impact compared to male users.

🎓 Academic Influence
Students with lower academic engagement tend to have higher addiction scores.

🛠️ Tools & Technologies Used
Power BI → Dashboard creation & visualization
DAX (Data Analysis Expressions) → Calculated measures and KPIs
Power Query → Data cleaning and transformation
Data Visualization Techniques → Charts, KPIs, slicers, and storytelling

🎯 Objective
The objective of this project is to analyze and visualize social media usage behavior to understand its relationship with addiction levels and mental health impact. This helps in identifying trends and promoting awareness about excessive social media usage.
